# The VC Brain — Vision

> Статус: DRAFT — прорабатывается с оператором (18.07.2026). Фиксация решений — через
> process-meetings в intel-базу.

## One-liner

Evidence-backed VC-мозг, который находит и скорит фаундеров ДО того, как их видит рынок,
и выдаёт инвестору decision-ready рекомендацию по чеку $100K за 24 часа — по merit, не по нетворку.

## Для кого (персоны — черновик, см. roadmap.md)

1. **Инвест-менеджер фонда** (primary, REQ-007) — живёт в дашборде, действует по memo.
2. **Агент/автоматизация** (primary наравне с человеком) — работает через API/CLI/MCP:
   получает данные, обновляет, ищет. Продуктовое требование оператора: сервис = сайт + API.
3. **Фаундер-заявитель** (secondary) — минимальная форма заявки (deck + company name + артефакты).

## Ставки (что делает нас не-«ещё-одним-скринером»)

- **Cold-start скоринг** фаундера без track record — главный тайбрейк рубрики (30% Data).
- **Evidence ledger**: каждый claim → источник → confidence; missing ≠ minus (REQ-003).
- **Оси не схлопываются** (REQ-002): Founder / Market / Idea-vs-Market + Trust, каждая с трендом.
- **Три защиты от наивных реализаций** (наши находки, RSK-002..004): provenance-check вместо
  code-volume, verbatim-слой против LLM-эхо-камеры, калибровка с учётом survivorship bias.
- **Agent-first доступ**: API/MCP как равноправный интерфейс — VC Brain встраивается в чужие
  агентные пайплайны (ни один из 9 OSS-референсов, кроме sieve-mcp, этого не делает).

## Не делаем (SCOPE-вердикты)

Sales-outreach механика, WhatsApp-активация, не-VC адаптации, downstream (portfolio/fund ops/exit).
